# ClothPOS - Python POS System

ClothPOS is a simple Point of Sale (POS) system developed to manage sales in a clothing store.
The system helps to record sales and generate bills for customers.

## Features

* Record product sales
* Generate customer bills
* Store sales data
* Simple and user-friendly interface

## Technologies Used

* Python
* Tkinter
* SQLite
